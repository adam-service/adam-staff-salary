# ADAM スタッフ報酬計算

ADAMスタッフ向けの業務委託報酬計算アプリ。

## 公開URL
https://adam-service.github.io/adam-staff-salary/

## 対応サービス
- **ADEN**（基本単価 18,000円 − ルーム料 8,000円 = 報酬 10,000円/回）
- **SUB**（基本単価 13,000円 − ルーム料 5,000円 = 報酬 8,000円/回）
- **NOIC**（広告経由ゲスト 6,000円/回 ／ 自己集客ゲスト 10,000円/回）
- **resync**（広告経由ゲスト 6,000円/回 ／ 自己集客ゲスト 10,000円/回）

セッション単価は各ゲスト個別、または月単位で変更可能。

## 機能
- PIN認証（スタッフ／管理者の2モード）
- 月別セッション数入力
- ゲスト管理（4サービス × チケット管理）
- レポート（サービス別小計＋合計報酬額）
- 月締め確定（管理者のみ解除可能）
- Google Apps Script + スプレッドシートによるクラウド同期
- 週次自動バックアップ（Google Drive）

## ファイル構成
- `index.html` — アプリ本体（GitHub Pages公開）
- `gas_code.js` — Google Apps Script バックエンド

## セットアップ手順
1. Google スプレッドシートを新規作成し、IDをコピー
2. 拡張機能 → Apps Script を開き、`gas_code.js` の内容を貼り付け
3. 上部の `SPREADSHEET_ID` をコピーしたIDに置き換え
4. デプロイ → 新しいデプロイ → ウェブアプリ
   - 次のユーザーとして実行: 自分
   - アクセスできるユーザー: 全員
5. デプロイ後のURLを、アプリの初期設定画面に入力

## 初期PIN
- スタッフ: `0000`
- 管理者: `0000`

ログイン後、設定画面でPINを変更してください。
